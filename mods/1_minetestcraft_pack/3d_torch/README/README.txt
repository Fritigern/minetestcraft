....................................................

1. extract the "3d_Torches_0.1" folder

2. Move the "3d_Torches" to the mod folder in your mintest game files

3. Start Minetest

4. under the "Singleplayer" tab press the Configure tab

5. enable "3d Torchest" then press Save

6. enjoy the mod
